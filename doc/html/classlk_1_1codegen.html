Detailed Description

Codegen produces bytecode from a tree of nodes.

Uses root node, a list_t node containing a vector of statements to produce an executable stack and lists of identifiers for interpretation by virtual machine. Original LK script text stored in srcpos_t are transfered to instr's, which are pushed onto stack.

Labels created at beginning of compound nodes (such as for loops or conditionals) contain information about position in stack of starting instruction.
