Detailed Description

Translates input into tokens.

Lexer takes as input an input_base and extracts tokens from the input LK string by identifying the structural pieces of the language. Lexer changes state as it traverses the input string when next() is called: the token type is returned and m_buf stores the string literal of the current token, which is accessed through text().
